## Architecture

VAK weapons (StA-55, Medi-Gun, etc.) do **not** use the legacy `Tool.SMG.Gun` LocalScript. They use:

```
PlayerGui.VAK_UI
├── ToolService (ModuleScript)   — Firing, Reload, Bind, CheckAmmo
├── gunModule (ModuleScript)     — FireFunction, rays, WeaponInfo
└── animModule (ModuleScript)

Tool (in Character/Backpack)
└── Client (ModuleScript)        — returns weapon instance u3
    setmetatable({}, ToolService)
```

### StA-55 defaults (`1_5819826_Client.ModuleScript.lua`)

| Field | Value | Role |
|-------|-------|------|
| `Name` | `"StA-55"` | Weapon id |
| `Ammo` | 50 | Current clip |
| `MaxAmmo` | 50 | Clip size |
| `RoF` | **0.1** | Seconds between shots (`wait(u13.RoF)` in Firing) |
| `ReloadTime` | **0.04** | Seconds per +1 ammo in reload loop |
| `IsAuto` | true | Hold-to-fire loop |
| `Out` | false → true on Equip | Equipped flag |

### Fire loop (`1_3512268_ToolService.ModuleScript.lua`)

Auto weapons (`IsAuto == true`):

1. `MouseClick` → `Firing(weapon)`
2. `while weapon.IsActive do`
3. `weapon.Ammo -= 1`
4. `weapon.Gun.FireFunction(...)`
5. **`wait(weapon.RoF)`** ← fire rate gate
6. On empty: `weapon:Reload()`

### Reload loop

```lua
while true do
    wait(p9.ReloadTime)
    p9.Ammo = p9.Ammo + 1
    if p9.Ammo == p9.MaxAmmo then break end
end
```

## Vape patch strategy (`5411969757.lua`)

### Discovery

- **VAK:** `require(tool.Client)` on tools in Character/Backpack; prefer `Out == true` (equipped).
- **Legacy:** `getsenv(tool.Gun)` LocalScript env.

### FireRate

- **VAK (primary):** `rawset(weapon, 'RoF', slider / 100)` every Heartbeat.
- **Fallback:** `wait` hook matches `0.1`, `0.09`, and `0.07–0.12` from Gun/ToolService/VAK_UI scripts.
- **Legacy:** wait hook + `u17 = true` on env.

### InfiniteAmmo

- **VAK:** `rawset(weapon, 'Ammo', weapon.MaxAmmo)` every Heartbeat (Firing decrements each shot).
- **Legacy:** `rawset(env, 'u6', true)` + `rawset(env, 'u13', 35)`.

### InstantReload

- **VAK:** `rawset(weapon, 'ReloadTime', 0)` + wrap `Reload` to instantly fill `Ammo`.
- **Legacy:** zero reload animation waits (`0.4`, `0.18`, `0.13`) + wrap `u25.Reload`.

## Safety constraints

- **Never** dot-access gunModule/ToolService env keys (`Mag` → `GetService("Mag")` error).
- Use **`rawget` / `rawset`** only on module env tables.
- **Never** overwrite `Requirement.OnInvoke` (server kicks with `"nope"`).
- **No `getgc`** scanning of metatable proxy tables.
